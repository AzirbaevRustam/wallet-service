# Wallet Service

REST API сервис для управления электронными кошельками.

## Описание

Микросервис предоставляет API для операций пополнения и списания средств с кошельков, 
а также для проверки баланса. Сервис разработан с учетом высоких нагрузок и конкурентного доступа.

## Технологический стек

- **Java 17**
- **Spring Boot 3**
- **PostgreSQL**
- **Liquibase** (миграции БД)
- **Docker & Docker Compose**
- **Testcontainers** (интеграционное тестирование)

## Основные функции

- ✅ Пополнение кошелька (DEPOSIT)
- ✅ Списание средств (WITHDRAW) 
- ✅ Проверка баланса
- ✅ Обработка конкурентных запросов
- ✅ Валидация входных данных
- ✅ Обработка ошибок
- ✅ Docker-контейнеризация

## API Endpoints

### POST /api/v1/wallet
Пополнение или списание средств с кошелька.

**Request:**
```json
{
  "walletId": "uuid",
  "operationType": "DEPOSIT|WITHDRAW", 
  "amount": 1000
}

GET /api/v1/wallets/{walletId}
Получение текущего баланса кошелька.
